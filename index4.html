Differences Between append, insertBefore, and insertAdjacentHTML:

1: append => it can take multiple argument..It is used to add one or more nodes and string as a children

2: insertBefore => it can take two argument..the first argument take a new node which we inserted in the document. And the second argument the node before which the new node will be inserted

3: insertAdjacentHTML => it can take two argument..the first argument take a postion that which position we want to add..And the second argument take a element node whic we want to add in that position
