Explain DOM Traversal:

DOM traversal refers to the process of navigating through the HTML document object model (DOM) to access or manipulate elements based on their relationship to other elements.It involves moving up, down, or sideways within the DOM tree to find specific elements, their children, siblings, or parent elements.

1: Parent Node
:- parentNode - This property allows you to access the parent node of a given element. :- parentElement - The parentElement property returns the parent element of the specified element

2: Child Node
:- childNodes - Returns collection of an element's child nodes including text and comment nodes :- children - Returns collection of an element's child element node do not include text and comment nodes :- firstChild - Returns first child node of an element.. It can be text or comment Node :- firstElementChild - Returns first element child node of an element :- lastChild - Returns last child node of an element.. It can be text or comment Node :- lastElementChild - Returns last element child node of an element

3: Siblind Node
:- previousSibling - Return previous node of same element.. It can be text or comment Node :- previousElementSibling - Return previous element node of same element :- nextSibling - Return next node of same element.. It can be text or comment Node :- nextElementSibling - Return next element node of same element
